# **A-6 Digital Tools Buying Website**

## **Description**
This is a modern and fully responsive digital tools marketplace built with React. Users can explore different digital products, view their features, and manage their cart easily.  

This project demonstrates clean component architecture, well-organized state management, and a smooth user experience enhanced with toast notifications.


## **Technologies Used**
- React.js  
- Tailwind CSS  
- DaisyUI  
- JavaScript (ES6+)  
- React-Toastify  
- JSON  


## **Features**

### **Product Cards & Cart Functionality**
Fully functional product browsing with visually appealing cards and a dynamic shopping cart. Users can add/remove items and see real-time updates. Duplicate additions are prevented with proper warning messages.

### **Smart Tab Switching**
Seamless toggle between **Products** and **Cart** sections using clean state management. By default, the Products section is displayed. The Cart section shows added items with real-time total calculation and individual remove functionality.

### **Real-time Cart Management with Toast Notification**
A complete cart system that displays selected products, allows individual item removal, calculates total price, and includes a **"Proceed to Checkout"** button that clears the cart. All actions are enhanced with user-friendly toast notifications.


## **Additional Highlights**
- Fully responsive design (mobile, tablet, and desktop)  
- Professional Navbar with live cart count  
- Eye-catching Banner and Stats section  
- Modern Footer with social media icons
